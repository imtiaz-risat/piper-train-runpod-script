# Create a new Pod

> Creates a new [Pod](#/components/schemas/Pod) and optionally deploys it.

## OpenAPI

````yaml POST /pods

  path: /pods
  method: post
  servers:
    - url: https://rest.runpod.io/v1
  request:
    security:
      - title: ApiKey
````
